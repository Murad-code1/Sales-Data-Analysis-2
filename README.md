# 🛒 Supermarket Sales - Exploratory Data Analysis

## 📌 Project Overview
This project performs exploratory data analysis (EDA) on a supermarket sales dataset to understand sales patterns across branches, product lines, and customer types.

The analysis focuses on identifying key trends in customer behavior, product performance, and sales distribution over time.

---

## 🎯 Objectives
- Analyze total sales and order distribution across:
  - Branch
  - Product Line
  - Customer Type
  - Gender
- Identify peak sales hours
- Determine top and bottom performing product categories
- Visualize key insights using charts

---

## 🛠️ Technologies Used
- Python
- pandas
- matplotlib
- seaborn

---

## 📊 Key Visualizations
- **Bar Chart:** Sales by Product Line  
- **Pie Chart:** Customer Type Distribution  
- **Line Chart:** Sales by Hour  
- **Heatmap:** Sales by Branch and Product Line  

---

## 📈 Key Insights
- Food and Beverages and Sports and Travel are the top-performing product categories.
- Sales peak during midday (13:00) and evening hours (19:00).
- Customer distribution between Member and Normal is nearly equal.
- Sales performance across branches is relatively balanced.

---

## 📂 Dataset
The dataset contains:
- 1000 rows and 17 columns  
- No missing values  
- Balanced distribution across branches
